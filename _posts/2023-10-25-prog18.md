# __part18. n^2 배열 자르기__

### _js ver 1.0_
```js 
function solution(n, left, right) {
    return Array(right - left + 1).fill(left).map((el, idx) => {
        return Math.max(Math.floor((el + idx) / n), (el + idx) % n) + 1
        });
}

```
